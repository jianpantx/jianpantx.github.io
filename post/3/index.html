在开发中遇到关于PHP_EOL的问题

2019-04-19

✔今天在开发中遇到PHP_EOL不换行的问题，之前只知道是用于换行，是php的一个预定义常量，对PHP_EOL的使用也没什么研究。今天把自己的查阅结果记录下来，加深一下记忆。

首先，PHP_EOL在不同平台输出的结果是不一样的

echo PHP_EOL;
//windows平台相当于    echo "\r\n";
//unix\linux平台相当于    echo "\n";
//mac平台相当于    echo "\r";

PHP_EOL的目的是自动为不同平台选择正确的字符，在终端上显示是可以正常的，但如果是在浏览器html则不能换行，是一个空格的效果。你需要在使用<pre>，或者使用php提供了nl2br这个字符串函数，能对字符串中所有新行之前插入<br>或<br />,效果看上去就是把\n、\r\n、\r替换，达到目标效果。

<?php
$a =  'hello'.PHP_EOL.'world';

echo $a;
echo '<hr />';
echo nl2br($a);
echo '<hr />';
echo '<pre>';
echo $a;
